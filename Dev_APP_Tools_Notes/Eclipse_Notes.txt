2) Aptana Eclipse Plug-in Installation link http://aptana.org/downloads/start
    Installing via Eclipse
    a)  Please copy the following Update Site URL to your clipboard and then follow the steps listed below to add this URL to your Available Software Sites list. Attempting to access this URL using your web browser will return an Access Denied error.
        -   http://download.aptana.com/studio3/plugin/install  
    b)  From the Help menu, select Install New Software... to open the Install New Software dialog.
    c) Paste the URL for the update site into the Work With text box, and hit the Enter (or Return) key.
    d) In the populated table below, check the box next to the name of the plug-in, and then click the Next button.
    e) Click the Next button to go to the license page.
    f) Choose the option to accept the terms of the license agreement, and click the Finish button.
       You may need to restart Eclipse to continue.

[edit] Finding Eclipse Version

for those times you need to verify that a plugin will work for you.
1) Select Help -> Search
2) Type About
3) From the Drop down results list select "About Eclipse"
4) From About Eclipse window select "Installation Details"
5) From Installation Details Eclipse Platform -> Eclipse Platform
The version will be in the column labeled version and look like this:
3.7.1.M20110909-1335
